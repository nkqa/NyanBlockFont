# NyanBlockFont（喵块体）

本人字体设计不是特别好，如有建议可以提出

## 声明
本字体使用日本字形，简体偏旁的字也采用日本字形，因此不适合用于教学等正式场合使用。

部分字符使用其它字体。

## 简介

**NyanBlockFont（喵块体 / 喵塊體 / ニャンブロックフォント）** 是一款由 **Meowco（喵可酱QAQ / 喵可醬QAQ / ねこかわ）** 制作的 CJK 字体。

该字体使用直线、简洁化的设计风格，目标是在中文、日文以及其他语言混排环境中提供清晰、可爱的显示效果。

## 使用方式

### Windows

1. 双击 `NyanBlockFont-Light.ttf`
2. 点击「安装」

### Linux

复制字体文件到：

```
~/.local/share/fonts/
```

然后刷新字体缓存：

```
fc-cache -fv
```

### macOS

打开字体文件，并使用字体册安装。
