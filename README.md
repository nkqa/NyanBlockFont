# NyanBlockFont（喵块体）

![NyanBlockFont Logo](logo.png)

## 简介

**NyanBlockFont（喵块体 / 喵塊體 / ニャンブロックフォント）** 是一款由 **Meowco（喵可酱QAQ / 喵可醬QAQ / ねこかわ）** 制作的 CJK 字体。

该字体使用直线、简洁化的设计风格，目标是在中文、日文以及其他语言混排环境中提供清晰、可爱的显示效果。

## 字符覆盖

NyanBlockFont 包含约 **9793 个字形（Glyph）**，支持多种 Unicode 字符区块。

主要覆盖：

| Unicode 区块                      | 覆盖情况         |
| ------------------------------- | ------------ |
| Basic Latin                     | 95 / 128     |
| Latin-1 Supplement              | 78 / 128     |
| Latin Extended                  | 部分支持         |
| Hiragana                        | 86 / 96      |
| Katakana                        | 92 / 96      |
| Bopomofo（注音符号）                  | 43 / 48      |
| CJK Unified Ideographs（中日韩统一汉字） | 9076 / 20992 |
| CJK Symbols and Punctuation     | 支持           |
| Halfwidth and Fullwidth Forms   | 支持           |
| CJK Extension C                 | 少量支持         |

字体主要面向中文、日文使用场景，同时包含基础拉丁字符和常用符号。

## 使用方式

### Windows

1. 双击 `NyanBlockFont-Light.ttf`
2. 点击「安装」

### Linux

复制字体文件到：

```
~/.local/share/fonts/
```

然后刷新字体缓存：

```
fc-cache -fv
```

### macOS

打开字体文件，并使用字体册安装。
